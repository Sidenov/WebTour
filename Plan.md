# Планирование автоматизации.
+ ## Перечень автоматизируемых сценариев:
### 1. Страница "Путешествие дня" доступна по адресу localhost:8080
> Шаги для проверки:
>   1. Открыть браузер и Перейти по адресу: localhost:8080
>   2. Проверить, что страница "Путешествие дня" открылась

### *2. Успешное одобрение заявки "Оплата по карте" при заполнении валидными данными и номером карты со статусом Approved*
> Шаги для проверки:
>   1. Открыть браузер и Перейти по адресу: localhost:8080
>   2. Нажать на кнопку "Купить"
>   3. Проверить, что название формы - "Оплата по карте"
>   4. В поле "Номер карты" ввести номер карты со статусом Approved: 4444 4444 4444 4441
>   5. В поле "Месяц" ввести валидное значение: 01
>   6. В поле "Год" ввести валидное значение: текущий год + 2 (например: 22)
>   7. В поле "Владелец" ввести валидное значение: Ivanov Ivan
>   8. В поле "CVC/CVV" ввести валидное значение: 111
>   9. Нажать кнопку "Продолжить"
> 
>   Ожидаемый результат: Появится сообщение "Успешно. Операция одобрена банком"

   - Java - является одним из самых популярных и распространенных языков программирования

   - Docker - система контейнеризации для запуска на одном компьютере несколько контейнеров
   - Git система контроля версий
